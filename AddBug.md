# How to add a bug

## EN

1. Fork this repo
2. Make a json file with schema
   ```json
   {
     "cdn": ["YOUR CDN Bug"],
     "sni": ["YOUR SNI BUG"]
   }
   ```
3. Place your file in `./resources/bugs/`
4. Create pull request
